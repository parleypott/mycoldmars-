What is a Lottie?

A Lottie is an animated vector graphic — like an animated SVG. It's a small text file that tells the browser exactly how shapes move, fade, and transform over time. The browser reads it and plays it perfectly at any size.

They're already everywhere: Airbnb, Duolingo, Spotify — those slick little animations in apps? Mostly Lotties. They're tiny (often under 50KB), loop seamlessly, and look sharp on any screen.

OmniLottie is a new AI model that generates these files from a text prompt. You say what you want, it produces the animation. The examples below are real Lotties — playing live in your browser right now.
