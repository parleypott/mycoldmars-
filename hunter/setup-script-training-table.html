Run this SQL in Supabase to create the global script training table

1 Go to Supabase SQL Editor
2 Paste and run this SQL: 
create table if not exists script_training (
  id uuid primary key default gen_random_uuid(),
  doc_count integer not null default 0,
  doc_titles jsonb default '[]',
  color_rules jsonb default '[]',
  sloppiness_patterns jsonb default '[]',
  structural_patterns jsonb,
  voice_style jsonb,
  visual_direction_style jsonb,
  script_context text,
  style_signature text,
  model text not null default 'gemini-2.5-flash',
  created_at timestamptz default now()
);

create index if not exists idx_script_training_created
  on script_training(created_at desc);
3 Click Run. Should say "Success. No rows returned."
4 Done! Go to Script Copilot
