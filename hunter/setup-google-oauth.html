GOOGLE OAUTH SETUP FOR SCRIPT COPILOT
01
Create a Google Cloud Project
Go to Google Cloud Console → New Project
Name it something like Hunter Script Copilot
Wait for it to create, then make sure it's selected.
02
Enable the APIs
Enable these two APIs:

Google Docs API — click ENABLE
Google Drive API — click ENABLE
03
Configure OAuth Consent Screen
Go to OAuth consent screen
Choose "External" user type, click Create
Fill in app name (Hunter), your email, and save
Under "Test users" add your Google account email
04
Create OAuth Credentials
Go to Credentials
Click "+ CREATE CREDENTIALS" → "OAuth client ID"
Application type: Desktop app
Name: Hunter CLI
Click Create

Copy the Client ID and Client Secret — you'll need them next.
05
Run the Setup Script
In your terminal, run:
node hunter/worker/google-auth-setup.mjs
Paste your Client ID and Secret when prompted.
The browser will open for Google consent — approve it.
The script will print your GOOGLE_REFRESH_TOKEN.

Add all three values to .env:
GOOGLE_CLIENT_ID=your-client-id
GOOGLE_CLIENT_SECRET=your-client-secret
GOOGLE_REFRESH_TOKEN=your-refresh-token
